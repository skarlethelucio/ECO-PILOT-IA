# EcoPilot AI

AI-powered telemetry for optimizing electric vehicle efficiency through physics-based modeling and real-time data.

## Run Locally

1. Clone the repository

```git clone https://github.com/your-username/ecopilot-ai.git```

``cd ecopilot-ai``

2. Install dependencies

``npm install``

3. Create environment variables

Create a .env file in the root directory and add:

``WEATHER_API_KEY=your_api_key_here``

``BACKBOARD_API_KEY=your_api_key_here``

⚠️ Note: API keys are required for weather data and AI analysis.

4. Start the server

``npm start``

Then open in your browser:

``http://localhost:3000``

## Overview

In electric racing, every watt matters.

EcoPilot AI transforms raw telemetry into actionable insights using physics modeling, real-time weather data, and AI — helping drivers make better energy decisions.

## 🏁 Inspiration

Built from experience in Borregos Racing.

We had data… but not intelligence behind it.

💡 Solution

🧠 Physics-based efficiency model (kWh/km + v² drag)

🌦️ Weather integration (temperature & wind)

🤖 AI-generated insights (analysis + driver tips)

📊 Interactive telemetry dashboard
